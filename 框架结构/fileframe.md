#B-JUI框架文件夹结构及文件说明
##子标题


```js
01|BJUI/
02├── js/ [-- 内置组件 JS --]
03│ ├── bjui-core.js [ 核心文件，包含框架的一些参数配置和初始化入口 ]
04│ ├── bjui-regional.zh-CN.js [ 国际化文件 ]
05│ ├── bjui-frag.js [ 一些组件的DOM碎片 ]
06│ ├── bjui-extends.js [ 包含部分jQuery方法及String方法 ]
07│ ├── bjui-basedrag.js [ 基础拖动 ]
08│ ├── bjui-slidebar.js [ 左侧导航菜单 ]
09│ ├── bjui-contextmenu.js [ 右键弹出菜单(主要应用在navtab 及 taskbar) ]
10│ ├── bjui-navtab.js [ 标签工作区 ]
11│ ├── bjui-dialog.js [ 弹窗 ]
12│ ├── bjui-taskbar.js [ 普通弹窗任务栏 ]
13│ ├── bjui-ajax.js [ 一些ajax方法 ]
14│ ├── bjui-alertmsg.js [ 提示框 ]
15│ ├── bjui-pagination.js [ 分页组件 ]
16│ ├── bjui-util.date.js [ 日期工具类 ]
17│ ├── bjui-datepicker.js [ 日期选择器 ]
18│ ├── bjui-ajaxtab.js [ 为bootstrap的tab增加ajax载入方式 ]
19│ ├── bjui-datagrid.js [ datagrid ]
20│ ├── bjui-tablefixed.js [ 固定表头表格 ]
21│ ├── bjui-tabledit.js [ 可编辑表格 ]
22│ ├── bjui-spinner.js [ 微调器 ]
23│ ├── bjui-lookup.js [ 查找带回 ]
24│ ├── bjui-tags.js [ 自动完成标签 ]
25│ ├── bjui-upload.js [ 上传组件 ]
26│ ├── bjui-theme.js [ 皮肤处理 ]
27│ ├── bjui-initui.js [ 框架初始化接口等 ]
28│ ├── bjui-plugins.js [ 监听bjui.initUI事件，使框架组件及部分插件初始化 ]
29│ ├── jquery-1.7.2.min.js
30│ └── jquery.cookie.js
31├── themes/ [-- CSS 样式 --]
32│ ├── default/ [ 皮肤 - 灰 ]
33│ ├── blue/ [ 皮肤 - 蓝 ]
34│ ├── green/ [ 皮肤 - 绿 ]
35│ ├── orange/ [ 皮肤 - 橙 ]
36│ ├── purple/ [ 皮肤 - 紫 ]
37│ ├── css/ [ 核心css目录 ]
38│ │ ├── FA/ [ FontAwesome 图标字体 ]
39│ │ ├── img/ [ zTree插件用图片 ]
40│ │ ├── style.css [ 核心css ]
41│ │ ├── bootstrap.min.css [ bootstrap css ]
42│ │ └── ie7.css [ 低版本IE警告提示css ]
43│ └── fonts/ [ bootstrap 图标字体 ]
44├── plugins/ [ -- 插件目录 -- ]
45│ ├── bootstrapSelect/ [ select美化 ]
46│ ├── colorpicker/ [ 颜色选择器 ]
47│ ├── dragsort/ [ 拖动排序 ]
48│ ├── icheck/ [ 单、复选框美化 ]
49│ ├── kindeditor_4.1.10/ [ 编辑器 ]
50│ ├── niceValidator/ [ 表单验证 ]
51│ ├── uploadify/ [ 上传组件-用于IE8-9 ]
52│ ├── swfupload/ [ kindeditor和uploadify共用的swf上传插件 ]
53│ ├── ztree/ [ zTree插件 ]
54│ ├── other/ [ 其他插件 ]
55│ │ └── jquery.autosize.js [ textarea高度自适应 ]
56│ └── bootstrap.min.js
57└── other/ [-- 其他 --]
58 ├── jquery.iframe-transport.js [ 为iframe方式提交表单提供支持（方便通过ajax方式提交file框） ]
59 ├── html5shiv.min.js [ for IE8-9 ]
60 └── respond.min.js [ for IE8-9 ]
```
